
# Stock Portfolio Tracker

## Description
A simple Python program that allows users to track their stock investments.  
Users enter stock names and quantities, and the program calculates the total investment based on predefined stock prices. Optionally, results are saved to a file.

## Features
- Predefined stock prices for 5 popular stocks: AAPL, TSLA, GOOGL, AMZN, MSFT.
- Accepts user input for stock names and quantities.
- Calculates total investment value.
- Saves portfolio details to a `portfolio.txt` file.
